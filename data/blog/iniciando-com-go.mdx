## Iniciando um Modulo

- Execute o Comando no seu terminal:

  ```
  go mod init helloworld
  ```

  {' '}

  <p>
    Abra seu Editor de Texto de preferencia no meu caso sera o Vs Code e crie uma arquivo chamado
    main.go.
    <br />
    Então crie uma função chamada main.
    <br />
    Ela deve ficar como o codigo abaixo.
  </p>

  ```go:main.go
  package main

  import "fmt"

  func main() {
    fmt.Println("Hello World")
  }
  ```

- Explicação
  - <strong>package main </strong> é o pacote no qual o Go vai compilar junto com suas dependencias
  - <strong>import "fmt" </strong> é aonde o go vai importar as funções para executar o print no seu
    terminal.
